Hangman-Game-.NET
=================

This a .Net webpage on which user can play the game called Hangman.
Basically a word guessing game in which a user is presented with a hidden word and he/she has to guess the word within a 
fixed number of attempts. The words to be guessed are randomly selected from the file called "TextFile" (each word has to be on a seperate line).

Guide:
1. Install Visual Studio 2010 or whatever is the latest.
2. Go onto File and then Open Project.
3. Browse to this project Folder.
4. CLick open and then click on debug in Visual Studio.
Enjoy
